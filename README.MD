#MoviesApp

## Descriptions
This application displays a list of movies from Kinopoisk (uses its free API). Clicking on a Movie opens a
poster, rating (Kinopoisk and IMDB) and the description of the movie.

## Skills
- Android Studio
- Java
- Retrifit
- MVVM
- Room Database
- SQLite
- Glide
